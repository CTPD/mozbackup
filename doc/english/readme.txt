MozBackup 1.5.2
===================================================

General
--------------------------------------------------
MozBackup is a simple utility for creating backups
of Mozilla applications like:

* Mozilla Firefox 
* Mozilla Thunderbird
* Mozilla Sunbird
* SeaMonkey / Mozilla Suite
* Flock
* Spicebird
* Netscape (7/9)
* Netscape Messanger
* Songbird
* PostBox
* Wyzo
 
It allows you to backup and restore bookmarks, mail, 
contacts, history, extensions, cache etc.

Supported versions:
* Firefox 1.0 or newer
* Thunderbird 1.0 or newer
* Sunbird 0.3 - 0.9
* Flock 1.0 - 2.6
* Postbox (Express) 1.0 or newer
* SeaMonkey 1.0a or newer
* Mozilla Suite 1.7 - 1.7.x
* Spicebird 0.4 - 0.8
* Songbird 1.0 or newer
* Netscape 7.x, 9.x
* Wyzo

Requirements
--------------------------------------------------
Windows 98 or higher (Windows 95 are not official
supported but maybe MozBackup works there too)
